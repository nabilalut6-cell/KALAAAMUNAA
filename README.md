# KALAAAMUNAA
Kalaamunaaa adalah aplikasi pembelajaran bahasa Arab berbasis web modern yang dikemas dengan visual interaktif ala Gaming, Dirancang khusus untuk melatih keterampilan berbicara (Maharah Kalam) siswa kelas X MA, aplikasi ini berfokus pada topik Al-Hayah Al-Yaumiyah, Al-Mihnah, dan Al-Usroh.
